# 使用说明

## 配置
### Shortcut 快捷键
* listModifier
列表快捷键修饰键
可以为 `alt` `ctrl` `win`/`meta` 
* listKeyType
列表快捷键类型
可以为 `char` `num`

### UI 界面
* maxListSize
最大列表项数, 默认值`9`

## 快捷键
* Enter
提交命令
* Up / Ctrl + K
向上移动选中项
* Down / Ctrl + J
向下移动选中项
